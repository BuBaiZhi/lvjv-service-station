## 📝 更新摘要

将用户中心从单页面详细列表模式改为主页面 + 二级页面的导航模式。用户进入用户中心时能一眼看到所有功能入口，需要查看具体内容时再进入二级页面。

## 🔄 需要同步到 HBuilderX 的操作

### 第 1 步：更新主页面
复制新的 `pages/user-center/index.vue` 到 HBuilderX 项目中

### 第 2 步：新增 5 个二级页面
在 HBuilderX 项目的 `src/pages/user-center/` 文件夹中新增以下 5 个文件：
- `order-list.vue`
- `publish-list.vue`
- `history-list.vue`
- `service-support.vue`
- `settings.vue`

### 第 3 步：刷新浏览器
1. 保存所有文件（Ctrl + S）
2. 清除浏览器缓存（Ctrl + Shift + Delete）
3. 刷新浏览器（F5）
