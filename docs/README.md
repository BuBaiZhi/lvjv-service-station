## 🔑 核心概念速查

### 全局主题系统

**问题**：硬编码颜色导致深色模式维护困难

**解决方案**：CSS 变量 + class selector

```wxss
.page.light {
  --bg-page: #f5f5f5;
  --bg-card: #fff;
}
.page.dark-mode {
  --bg-page: #1a1a1a;
  --bg-card: #2a2a2a;
}

.page { background: var(--bg-page); }
```

**优势**：一处定义，全局生效 ✅

### 深色模式实现

**WXML 绑定**：
```xml
<view class="page {{theme === 'dark' ? 'dark-mode' : ''}}">
```

**WXSS 响应**：
```wxss
.page.light { /* 浅色样式 */ }
.page.dark-mode { /* 深色样式 */ }
```

**完全覆盖**：背景、卡片、文字、边框、图标全部适配

### 老人友好版

**放大规则**：
- 字体 `+8px`
- 按钮 `+10rpx` 高度
- 间距 `+10rpx`
- 图片 `+100rpx` 尺寸

**实现方式**：
```wxss
.elderly-mode .tab-item {
  font-size: 40rpx;  /* 原 30rpx + 8px 放大 */
}
```
