# 服务与支持页面设计文档

## 📋 页面概述

**页面名称**：服务与支持  
**路由**：`/pages/user-center/service-support`  
**功能**：提供客服中心、服务咨询、常见问题等帮助信息  
**优先级**：⭐⭐ 辅助功能

## 🎯 功能需求

### 核心功能

1. **服务菜单展示**
   - 显示各类服务选项
   - 支持点击跳转
   - 显示服务描述
   - 显示联系方式

2. **服务类型**
   - 客服中心 - 在线客服

3. **联系方式**
   - 显示电话号码
   - 显示邮箱地址
   - 显示工作时间
   - 显示在线状态

## 🔧 技术实现

### 状态管理

```typescript
// 页面状态
const services = ref<Service[]>([])        // 服务列表
const contactInfo = ref<ContactInfo[]>([]) // 联系方式
const loading = ref(false)                 // 加载状态
```

### 方法

```typescript
// 加载服务列表
const loadServices = async () => {
  loading.value = true
  try {
    // 模拟 API 调用
    await new Promise(resolve => setTimeout(resolve, 500))
    services.value = mockServices
    contactInfo.value = mockContactInfo
  } finally {
    loading.value = false
  }
}

// 处理服务操作
const handleServiceAction = (service: Service) => {
  switch (service.action) {
    case 'navigate':
      if (typeof uni === 'undefined') {
        alert(`导航到: ${service.title}`)
      } else {
        uni.navigateTo({
          url: service.actionUrl || ''
        })
      }
      break
    case 'contact':
      contactService(service)
      break
    case 'link':
      openLink(service.actionUrl || '')
      break
  }
}

// 联系客服
const contactService = (service: Service) => {
  if (typeof uni === 'undefined') {
    alert(`联系: ${service.title}`)
  } else {
    // 打开客服对话框
    uni.showToast({ title: '正在连接客服...', icon: 'loading' })
  }
}

// 打开链接
const openLink = (url: string) => {
  if (typeof uni === 'undefined') {
    window.open(url, '_blank')
  } else {
    uni.openURL({
      url: url
    })
  }
}

// 复制联系方式
const copyContactInfo = (value: string) => {
  if (typeof uni === 'undefined') {
    // 浏览器环境
    navigator.clipboard.writeText(value).then(() => {
      alert('已复制到剪贴板')
    })
  } else {
    // uni-app 环境
    uni.setClipboardData({
      data: value,
      success: () => {
        uni.showToast({ title: '已复制', icon: 'success' })
      }
    })
  }
}

// 返回
const goBack = () => {
  if (typeof uni === 'undefined') {
    window.history.back()
  } else {
    uni.navigateBack()
  }
}

// 页面加载
onMounted(() => {
  loadServices()
})
```
