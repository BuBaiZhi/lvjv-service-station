# UI 设计方案

## 🎨 设计理念

**主题**：乡村生活、自然、温暖、亲切
**风格**：简洁现代 + 乡村温暖感
**目标用户**：村民和数字游民

---

## 🌈 色彩系统

### 主色调
- **主绿色**：`#4CAF50` - 自然、生机、信任
- **鹅黄色**：`#FDD835` - 温暖、阳光、活力

## 🎯 组件设计

### 1. 用户信息卡片 (UserCard)

**设计特点：**
- 渐变背景：绿色 → 浅绿色
- 卡片式设计，圆角 16px
- 头像带白色边框（3px）
- 社区数据用分割线分隔

**布局：**
```
┌─────────────────────────────────┐
│  [头像]  昵称                    │
│          身份标签 性别            │
│          个人简介...             │
│                    [编辑资料]    │
├─────────────────────────────────┤
│  发帖数  │  评论数  │  获赞数    │
│   24    │   156   │    89      │
└─────────────────────────────────┘
```

**色彩应用：**
- 背景：绿色渐变 (#4CAF50 → #81C784)
- 文字：白色
- 按钮：半透明白色背景，悬停时加深

---

### 2. Tabs 切换组件

**设计特点：**
- 简洁的分段控制器
- 选中状态：绿色背景 + 白色文字
- 未选中状态：浅灰色背景 + 深灰色文字
- 圆角 8px

**样式：**
```
┌──────────┬──────────┬──────────┐
│ ✓ 全部   │  房源    │  活动    │
└──────────┴──────────┴──────────┘
  绿色背景    浅灰背景   浅灰背景
```

---

### 3. 列表卡片 (OrderCard / PublishCard)

**设计特点：**
- 白色背景，圆角 12px
- 阴影：0 4px 6px rgba(0,0,0,0.1)
- 左侧图片 + 右侧信息布局
- 状态标签用不同颜色区分

**布局：**
```
┌─────────────────────────────────┐
│ [图]  标题                  [状态]│
│ 100x │  描述信息...              │
│ 100  │  ¥价格    2024-02-10     │
│      │                  [操作]   │
└─────────────────────────────────┘
```

**状态标签颜色：**
- 待确认：黄色背景 (#FFF3CD) + 深黄文字
- 已确认：蓝色背景 (#D1ECF1) + 深蓝文字
- 已完成：绿色背景 (#D4EDDA) + 深绿文字
- 已取消：红色背景 (#F8D7DA) + 深红文字

## 💡 设计建议

### 优先级
1. **必做**：主色调（绿色 + 鹅黄色）、基础排版、卡片设计
2. **重要**：按钮设计、状态标签、空状态
3. **可选**：深色模式、高级动画、微交互

### 实现步骤
1. 更新 `styles/variables.scss` 中的颜色变量
2. 更新各组件的样式，应用新色彩系统
3. 测试不同屏幕尺寸的显示效果
4. 收集反馈并迭代优化

### 一致性检查清单
- [ ] 所有按钮使用统一的绿色
- [ ] 所有卡片使用统一的圆角和阴影
- [ ] 所有文字使用统一的字体大小和颜色
- [ ] 所有间距使用统一的间距系统
- [ ] 所有过渡使用统一的动画时间
