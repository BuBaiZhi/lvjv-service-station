# 一键启动指南

## 🎯 最简单的方式

### 只需 3 步

#### 第 1 步：下载 HBuilderX
https://www.dcloud.io/hbuilderx.html

#### 第 2 步：创建项目
```
HBuilderX → 文件 → 新建 → 项目
选择 uni-app，Vue 3 + TypeScript
项目名称：user-center
```

#### 第 3 步：复制文件到项目根目录
将这些文件夹复制到 **HBuilderX 项目的根目录** 中：
- pages/
- components/
- stores/
- services/
- mock/
- types/
- styles/
- utils/

**复制位置**：`HBuilderX 项目根目录/`（不是 src 文件夹）

**注意**：HBuilderX 创建的项目没有 src 文件夹，直接复制到根目录即可。

详见：`项目结构对比.md`

---

## 🚀 运行项目

### 安装依赖（第一次）
```bash
# 在 HBuilderX 终端中运行
npm install pinia uni-ui
```

### 修改 main.ts（重要！）
```
打开 src/main.ts，添加 Pinia 初始化代码
详见：main.ts修复指南.md
```

### 启动项目
```
HBuilderX → 运行 → 运行到浏览器 → Chrome
```

浏览器会自动打开，显示项目效果。
