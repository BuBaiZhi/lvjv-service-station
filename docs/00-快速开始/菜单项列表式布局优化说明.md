# 菜单项列表式布局优化说明

**修改时间**：2026年2月12日
**修改版本**：v1.2.2 → v1.2.3
**修改状态**：✅ 已完成

---

## 📋 修改概述

将菜单项从网格布局改为列表式布局，每个功能占一整行，左边是图标，中间是文字，右边是箭头。这样更清晰、更易于使用。

### 修改前后对比

| 方面 | 修改前 | 修改后 |
|------|------|------|
| 菜单布局 | 5 列横向排列 | 列表式纵向排列 |
| 每行内容 | 图标 + 文字（居中） | 图标 + 文字 + 箭头（左中右） |
| 卡片高度 | 100px | 50px |
| 卡片宽度 | 均匀分布 | 100% 宽度 |
| 图标大小 | 32px | 24px |
| 文字大小 | 14px | 16px |
| 文字对齐 | 居中 | 左对齐 |
| 视觉风格 | 卡片式 | 列表式 |

---

## 🔧 修改内容

### 修改的文件

**文件**：`pages/user-center/index.vue`

### 具体修改

#### 1. 菜单容器改为纵向排列

**修改前**：
```scss
.modules-container {
  display: flex;
  flex-direction: row;
  gap: $spacing-md;
  width: 100%;
  padding: $spacing-lg;
  overflow-x: auto;
  justify-content: space-between;
}
```

**修改后**：
```scss
.modules-container {
  display: flex;
  flex-direction: column;
  gap: $spacing-sm;
  width: 100%;
  padding: $spacing-lg;
}
```

**修改说明**：
- ✅ 改为 `flex-direction: column`（纵向排列）
- ✅ 改为 `gap: $spacing-sm`（8px，更紧凑）
- ✅ 移除 `overflow-x: auto` 和 `justify-content: space-between`

#### 2. 菜单卡片改为列表式

**修改前**：
```scss
.module-card {
  padding: $spacing-md;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  min-height: 100px;
  min-width: 80px;
  flex: 1;
}
```

**修改后**：
```scss
.module-card {
  padding: $spacing-md $spacing-lg;
  display: flex;
  flex-direction: row;
  align-items: center;
  justify-content: space-between;
  min-height: 50px;
}
```

**修改说明**：
- ✅ 改为 `flex-direction: row`（横向排列）
- ✅ 改为 `justify-content: space-between`（左中右分布）
- ✅ 改为 `align-items: center`（垂直居中）
- ✅ 减小 `min-height` 从 100px 到 50px
- ✅ 移除 `min-width` 和 `flex: 1`

#### 3. 图标调整

**修改前**：
```scss
.module-icon {
  font-size: 32px;
  margin-bottom: $spacing-sm;
}
```

**修改后**：
```scss
.module-icon {
  font-size: 24px;
  margin-right: $spacing-md;
  flex-shrink: 0;
}
```

**修改说明**：
- ✅ 减小 `font-size` 从 32px 到 24px
- ✅ 改为 `margin-right`（左右排列）
- ✅ 添加 `flex-shrink: 0`（防止压缩）

#### 4. 文字调整

**修改前**：
```scss
.module-name {
  font-size: $font-size-sm;
  font-weight: $font-weight-medium;
  text-align: center;
}
```

**修改后**：
```scss
.module-name {
  font-size: $font-size-base;
  font-weight: $font-weight-medium;
  text-align: left;
  flex: 1;
}
```

**修改说明**：
- ✅ 增大 `font-size` 从 14px 到 16px（更易读）
- ✅ 改为 `text-align: left`（左对齐）
- ✅ 添加 `flex: 1`（占据剩余空间）

#### 5. 箭头调整

**修改前**：
```scss
.module-arrow {
  position: absolute;
  top: $spacing-sm;
  right: $spacing-sm;
  font-size: $font-size-base;
}
```

**修改后**：
```scss
.module-arrow {
  font-size: $font-size-lg;
  margin-left: $spacing-md;
  flex-shrink: 0;
}
```

**修改说明**：
- ✅ 移除 `position: absolute`（改为相对定位）
- ✅ 改为 `margin-left`（右对齐）
- ✅ 增大 `font-size` 从 16px 到 18px
- ✅ 添加 `flex-shrink: 0`（防止压缩）

#### 6. 悬停效果调整

**修改前**：
```scss
&:hover {
  box-shadow: $shadow-md;
  transform: translateY(-2px);
}
```

**修改后**：
```scss
&:hover {
  box-shadow: $shadow-md;
  background-color: $bg-secondary;
}
```

**修改说明**：
- ✅ 改为改变背景色而不是移动位置
- ✅ 更符合列表式菜单的交互风格
