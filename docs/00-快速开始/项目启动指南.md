---

## 📋 启动步骤（3 步）

### 第 1 步：下载并安装 HBuilderX

访问：https://www.dcloud.io/hbuilderx.html

选择"标准版"下载安装（约 200MB）

---

### 第 2 步：在 HBuilderX 中创建项目

```
1. 打开 HBuilderX
2. 文件 → 新建 → 项目
3. 选择 uni-app
4. 项目名称：user-center
5. 选择 Vue 3 + TypeScript 模板
6. 点击创建
```

项目会自动创建在 HBuilderX 的工作区中。

---

### 第 3 步：复制项目文件

将我们已经创建的文件复制到项目中：

```
复制这些文件夹到项目根目录：
├── pages/
├── components/
├── stores/
├── services/
├── mock/
├── types/
├── styles/
└── utils/
```

**具体操作**：
1. 在文件管理器中打开我们的项目文件夹
2. 复制上述文件夹
3. 粘贴到 HBuilderX 创建的项目目录中
4. 在 HBuilderX 中刷新项目（F5）

---

## 🚀 运行项目

### 在浏览器中预览（最快）

```
1. 在 HBuilderX 中打开项目
2. 点击菜单：运行 → 运行到浏览器 → Chrome
3. 浏览器会自动打开，显示项目效果
```

**优点**：
- 最快（2-3 秒）
- 支持热更新（修改代码自动刷新）
- 易于调试

---

### 在小程序模拟器中预览（真实环境）

```
1. 在 HBuilderX 中打开项目
2. 点击菜单：运行 → 运行到小程序模拟器 → 微信开发者工具
3. 微信开发者工具会自动打开，显示项目效果
```

**优点**：
- 真实的小程序环境
- 支持所有小程序特性

## 📦 安装依赖

在 HBuilderX 中打开终端：

```bash
# 方式 1：在 HBuilderX 中打开终端
菜单 → 工具 → 终端 → 新建终端

# 然后运行
npm install pinia uni-ui
```

或者在项目目录中打开 PowerShell：

```bash
# 右键点击项目文件夹 → 在此处打开 PowerShell

```

## ✅ 验证项目是否正常运行

打开浏览器后，检查以下内容：

- [ ] 页面加载成功（没有空白）
- [ ] 用户卡片显示绿色渐变背景
- [ ] 订单列表显示数据
- [ ] 没有浏览器控制台错误

按 F12 打开开发者工具，查看 Console 标签页是否有错误。

## 🐛 如果遇到问题

### 问题 1：页面空白

**解决**：
1. 按 F12 打开开发者工具
2. 查看 Console 标签页的错误信息
3. 根据错误信息调整代码

### 问题 2：找不到模块

**解决**：
1. 确保所有文件都复制到了正确的位置
2. 检查文件路径是否正确
3. 重启 HBuilderX

### 问题 3：样式不生效

**解决**：
1. 确保 `styles/variables.scss` 已创建
2. 检查组件中是否正确导入了样式
3. 清除浏览器缓存（Ctrl + Shift + Delete）

### 问题 4：npm 安装失败

**解决**：
```bash
# 清除缓存
npm cache clean --force

# 重新安装
npm install pinia uni-ui
```
