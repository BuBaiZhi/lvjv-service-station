# 项目结构说明

## 📁 项目目录结构

```
user-center/
├── docs/                              # 📚 所有文档都在这里
│   ├── README.md                      # 文档总览
│   ├── 00-快速开始/

│   │   └── UI组件使用指南.md
│   ├── 02-项目管理/
│   │   ├── 项目更新指南.md
│   │   └── 更新速查表.md
│   └── 03-项目规划/
│       ├── 用户中心功能设计.md
│       └── 开发步骤计划.md
│
├── pages/                             # 页面文件
│   └── user-center/
│       └── index.vue                  # 用户中心主页面
│
├── components/                        # 可复用组件
│   ├── UserCard.vue                   # 用户信息卡片
│   ├── OrderList.vue                  # 订单列表
│   ├── PublishList.vue                # 发布内容列表
│   ├── HistoryList.vue                # 历史记录列表
│   ├── ServiceSupport.vue             # 服务支持
│   └── SettingsList.vue               # 设置列表
│
├── stores/                            # Pinia 状态管理
│   └── user.ts                        # 用户状态 store
│
├── services/                          # API 服务层
│   └── user.ts                        # 用户相关 API
│
├── mock/                              # Mock 数据
│   └── userData.ts                    # 用户 Mock 数据
│
├── types/                             # TypeScript 类型定义
│   └── user.ts                        # 用户相关类型
│
├── styles/                            # 全局样式
│   └── variables.scss                 # 设计系统变量
│
├── utils/                             # 工具函数
│   └── storage.ts                     # 本地存储工具
│
├── README.md                          # 项目总览
└── package.json                       # 项目配置
```
